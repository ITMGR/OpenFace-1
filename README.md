# OpenFace
Face recognition solution that can be used as a REST web service

This is basically a wrapper for [cmusatyalab/openface]https://github.com/cmusatyalab/openface Docker solution. It contains scripts for automatic environment setup and startup.( 
# Requirements
- Docker installed on Windows, Mac or Linux
